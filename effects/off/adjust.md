| None                          |   |
| Adjust By Value               |   |
| Adjust By Percentage          |   |
| Set Minimum                   |   |
| Set Maximum                   |   |
| Set Range                     |   |
| Shift With Wrap By Value      |   |
| Shift With Wrap By Percentage |   |
| Prevent Range                 |   |
| Reverse                       |   |
